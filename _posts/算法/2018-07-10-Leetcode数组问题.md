## 7. Reverse Integer
**给你一个整数，翻转它**
```
Example 1:

Input: 123
Output: 321
Example 2:

Input: -123
Output: -321
Example 3:

Input: 120
Output: 21]
]
```
```
要注意越界的问题，long和int
```
```
class Solution {
    public int reverse(int x) {
        long a = 0;
        while(x !=0){
            a = x%10 + a*10;
            x = x/10;
        }
        if(x<0)
            a = 0 - a;
        return (int)a == a ? (int)a : 0;
    }
}
```
## 9. Palindrome Number(回文)
**给你一个整数，判断其是不是回文数**
```
Example 1:

Input: 121
Output: true
Example 2:

Input: -121
Output: false
```
```
class Solution {
    public boolean isPalindrome(int x) {
        if(x < 0)
            return false;
        int a = 0;
        int b = x;
        while(b != 0){
            a = a*10 + b%10;
            b = b/10;
        }
        
        return a ==x;
    }
}
```
